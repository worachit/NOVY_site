ฐานข้อมูล Big Data ของภาพถ่ายต้นมันสำปะหลังจากโดรน เพื่อการพัฒนาระบบการเรียนรู้ปัญญาประดิษฐ์

รายละเอียดฐานข้อมูล

Cassava เป็นฐานข้อมูล Big Data ของภาพถ่ายแปลงต้นมันสำปะหลังมุมสูงจากโดรน ชุดข้อมูล Big Data ประกอบไปด้วยชุดรูปภาพมุมสูงจากโดรน และ Annotatoin บอกถึงตำแหน่ง และโรคของต้นมันสำปะหลังกำกับในรูปแบบของ COCO API ฐานข้อมูลหมาะสำหรับการทำ Object Detection และ Image Classification เพื่อการทำระบบการเรียนรู้ของเครื่องจักร (Machine learning) เพื่อจำแนกโรคต่างๆในมันสัมปะหลัง เช่น โรคใบด่าง หรือเพื่อพัฒนาระบบการวางแผนเพื่อบินโดรนการเกษตรอัตโนมัติ
